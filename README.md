# logcat-reflection [![Build Status](https://travis-ci.org/zulhilmizainuddin/logcat-reflection.svg?branch=master)](https://travis-ci.org/zulhilmizainuddin/logcat-reflection)
Android logcat using reflection

## Build
Run

    gradlew jar
    
## Usage

```java
Log.v("tag", "log");
Log.d("tag", "log");
Log.i("tag", "log");
Log.w("tag", "log");
Log.e("tag", "log");
```
